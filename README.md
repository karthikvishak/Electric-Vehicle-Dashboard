# 🚗⚡ Electric Vehicle Dashboard

An interactive data analytics project built in **Microsoft Excel** and **Power BI**, analyzing Electric Vehicle (EV) adoption trends across the United States using a dataset of **114,595 vehicles**.

---

## 📊 Project Overview

This project explores real-world EV registration data to uncover trends in adoption rates, manufacturer dominance, state-wise distribution, and clean fuel eligibility. It features two dashboards — one built in Excel and one in Power BI — each delivering actionable insights through visual storytelling.

## 🔍 Key Insights

- 🔋 **76.6%** of registered EVs are Battery Electric Vehicles (BEV); **23.4%** are Plug-in Hybrids (PHEV)
- 🏆 **Tesla** leads all manufacturers with **52,672 vehicles** (~46% market share)
- 🗺️ **Washington State** dominates registrations with **114,312 vehicles**
- 📈 EV adoption **peaked in 2022** with 27,077 new registrations — up from just 23 in 2008
- ⚡ **50.7%** of vehicles qualify as Clean Alternative Fuel Vehicle (CAFV) eligible
- 🚘 **Jaguar** has the highest average electric range at **206 miles**; overall average is **105 miles**
- 🏅 Top 3 manufacturers by count: Tesla (52,672) → Nissan (12,839) → Chevrolet (10,273)
