# Electric Vehicle Dashboard

## Project Overview
This Excel dashboard provides insights into Electric Vehicle (EV) adoption and trends using interactive visualizations.

## Features
- Total EV registrations
- EV type analysis
- State-wise distribution
- Manufacturer analysis
- Interactive slicers and filters
- KPI tracking

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
